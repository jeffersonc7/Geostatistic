# Geoestadística (Python + Jupyter)

Este repositorio reúne material de **Geoestadística** desarrollado en **Python** mediante **Jupyter Notebooks**. Incluye:

- **Teoría** y notas de apoyo (conceptos, supuestos e interpretación).
- **Proyecto aplicado** con flujo completo: preparación de datos, estimación, diagnóstico.
- **Bases de datos** utilizadas en la teoría y en el proyecto.
- **Modelos econométricos espaciales** y técnica no paramétrica (con comparación de resultados).
- **Visualizaciones** y **pruebas/diagnósticos** para evaluar dependencia espacial y comportamiento de los residuales.

## Estructura del repositorio
- __Theory__ : notebooks y notas teóricas.
- __Proyecto__ → implementación de técnicas geoestadísticas.
- __Bases de datos__ → para proyecto y teoría.

## Tecnologías
- Python
- Jupyter Notebook
